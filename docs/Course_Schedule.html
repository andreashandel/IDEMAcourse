IDEMA Fall 2020 - Course Schedule

Andreas Handel

2020-08-11

Overview

This is the current course schedule for the class. Things will likely change somewhat as the semester progresses. Check regularly for updates.

The External Sources column only lists major external sources. Material provided on this website, as well as other smaller assignments embedded or linked in the course documents (e.g. videos, papers) are not listed. This is only meant as a quick reference, the course materials provide more detail and are the definite source in case of discrepancies.

Abbreviations used are:

  • IDEMAB = Infectious Disease Epidemiology - A model-based approach book. Numbers refer to the chapter.
  • DSAIDE = Dyanamical Systems Approach to Infectious Disease Epidemiology - R package. Numbers refer to the number of the app.

The Assessments column lists items that need to be submitted and that will be graded. Those are described in detail in the assessment document for each module. Unless otherwise specified, submission deadline is the module end date (Friday) 5pm.
